# Housing Brokers System

A full-stack web application designed to help users search for rental properties, manage listings, and handle bookings.

## Features
- User authentication (admin & agents)
- Property listing and management
- Booking and availability system
- Reporting and verification system

## Tech Stack
- PHP (backend)
- JavaScript (frontend)
- MySQL (database)
- Blade templates
